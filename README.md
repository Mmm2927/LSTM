## 개요

1. 육아 기록 및 건강 관리가 가능한 스마트 융가 수첩의 필요성
2. 수면 중 갑작스럽게 사망하는 영아 돌연사 증후군 예방

&Rightarrow; 생활 기록 및 홈캠을 이용한 수면 자세 판별 기능을 제공하여 부모들의 육아 스트레스를 줄여주는 통합 육아 수첩 애플리케이션을 개발하고자 함
